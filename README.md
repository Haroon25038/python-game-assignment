CSC-40072 Assessment: Simple Python Game

# Simple Python Game
# Overview

- This is a simple Python game called Guess the Number, developed as part of my programming assignment.
- It demonstrates the use of variables, loops, conditional statements, and functions to create an interactive experience.
- A second version of the game was created with a Graphical User Interface (GUI) using Tkinter.

# Features

- Interactive text-based gameplay

- Randomized number generation for replayability

- Simple and clean Python structure

- GUI version for better user interaction

- Focus on logic, functions, and user input

# Files Included

- main.py — Console version of the game

- guess_the_number_gui.py — GUI version of the game using Tkinter

- README.md — This file explaining the purpose of the project

# How to Run

- Make sure you have Python 3.8 or later installed.

- Download all files (main.py, guess_the_number_gui.py, and README.md) from the GitHub repository.

- Open the folder in VS Code, PyCharm, or any Python IDE.

# To play the console version, run:
- python main.py

# To play the GUI version, run:
- python guess_the_number_gui.py
