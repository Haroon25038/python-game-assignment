# Simple Python Game

## Overview
This is a simple Python game developed as part of my programming assignment.  
It demonstrates the use of variables, loops, conditional statements, and functions to create an interactive experience.

## Features
- Interactive text-based gameplay  
- Randomized outcomes for replayability  
- Simple and clean Python structure  
- Focus on logic, functions, and user input

## Files Included
- **main.py** — The main Python file that runs the game  
- **README.md** — This file explaining the purpose of the project  

## How to Run
1. Make sure you have **Python 3.8 or later** installed.  
2. Download both files (`main.py` and `README.md`) from the GitHub repository.  
3. Open the folder in VS Code, PyCharm, or any Python IDE.  
4. Run the game.
